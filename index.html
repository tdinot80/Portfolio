Roblox Studio



I started working in Roblox Studio in 2019 as the first program I used to create games. it remains the program I have the most experience using and there is nothing I can't do given enough time. I've worked mainly with programing, but due to having knowledge of the entire platform I frequently use it to my advantage turning what most would see as Roblox's disadvantages into advantages.







One of my greatest achievements is creating a highly optimized terrain generation system using perlin noise and marching cubes.

Generating terrain in a 64,000x64,000 stud grid around the player, meaning the nearest edge from the player is always over 9,000 meters away. The generation includes caves and has no upper or lower height limit.
